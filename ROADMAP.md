# Game Partner Skill - 项目路线图

## 🎯 愿景

打造全球首款开源的、完全本地化的智能游戏伴侣,让每位单机玩家都能享受"有人陪玩"的温暖体验。

---

## 📅 开发阶段

### Phase 0: 技术预研 (已完成 ✅)
- [x] 技术栈调研
- [x] 可行性验证
- [x] 需求文档编写
- [x] 开发计划制定

### Phase 1: MVP (2026-01-28 ~ 2026-02-27) 🚧
**目标**: 验证核心概念,完成可演示的 Demo

#### 核心功能
- [x] Tauri 应用框架
- [ ] 截屏 + OCR 识别
- [ ] Wiki 爬虫与向量库
- [ ] RAG + LLM 提示生成
- [ ] TTS 语音播报
- [ ] 基础弹幕 UI

#### 支持游戏
- [ ] 《艾尔登法环》(Elden Ring)
- [ ] 《博德之门3》(Baldur's Gate 3) - 可选

#### 交付物
- [ ] Windows 安装包 (.exe)
- [ ] 演示视频 (3-5 分钟)
- [ ] 技术文档

---

### Phase 2: Beta 版本 (2026-03 ~ 2026-05)
**目标**: 完善功能,扩展游戏支持,收集用户反馈

#### 新功能
- [ ] 自动游戏识别 (无需手动选择)
- [ ] 记忆系统 (记录玩家进度/偏好)
- [ ] 高级语音交互 (ASR 语音唤醒)
- [ ] 自定义 AI 人设 (甜美/御姐/少年)
- [ ] 数据导出 (游戏日志/统计)

#### 性能优化
- [ ] OCR 多引擎对比 (Tesseract + PaddleOCR)
- [ ] 本地 LLM 优化 (量化模型)
- [ ] 向量库查询加速 (HNSW 索引)
- [ ] 内存占用优化 (< 300 MB)

#### 支持游戏扩展
- [ ] 《黑暗之魂3》
- [ ] 《塞尔达传说:旷野之息》(模拟器)
- [ ] 《巫师3》
- [ ] 社区投票决定更多游戏

#### 平台扩展
- [ ] macOS 支持
- [ ] Linux 支持 (Steam Deck 友好)

---

### Phase 3: 1.0 正式版 (2026-06 ~ 2026-08)
**目标**: 产品化,构建社区,探索商业模式

#### 社区功能
- [ ] **Wiki 库共享**
  - 用户上传自己爬取的 Wiki
  - P2P 分享或中心化仓库
- [ ] **自定义模板**
  - OCR 区域自定义 (适配不同分辨率)
  - 弹幕样式自定义
- [ ] **插件系统**
  - 第三方开发者可扩展功能
  - 示例: Mod 提示、存档管理

#### 高级功能
- [ ] **计算机视觉识别**
  - 替代 OCR,直接识别游戏场景
  - 使用轻量 CNN 模型 (如 MobileNet)
- [ ] **实时战斗分析**
  - 检测血量、技能 CD
  - 提供战术建议
- [ ] **成就追踪**
  - 集成 Steam 成就 API
  - 完成时播报鼓励

#### 商业化探索 (可选)
- [ ] 付费高级音色 (专业配音演员)
- [ ] 云端 LLM 服务 (更智能的提示)
- [ ] 企业授权 (游戏工作室合作)

#### 营销推广
- [ ] Steam Workshop 发布
- [ ] B 站/YouTube 演示视频
- [ ] Reddit/Hacker News 宣传
- [ ] 游戏媒体投稿 (IGN, GameSpot)

## 🔮 长期愿景 (2026+)

### 多模态交互
- [ ] VR/AR 支持 (在游戏内看到 AI 伴侣虚拟形象)
- [ ] 手势控制 (摄像头检测手势唤醒)
- [ ] 情绪识别 (分析玩家语音情绪,调整回应风格)

### 跨游戏生态
- [ ] **通用游戏助手**
  - 自动适配任意游戏 (基于 CV + LLM)
- [ ] **跨游戏记忆**
  - 例: "你在《艾尔登法环》里很擅长闪避,在《黑魂》里也试试"
- [ ] **社交功能**
  - 好友系统,分享游戏时刻
  - AI 伴侣可以"见证"你的游戏历程

### AI Agent 进化
- [ ] **主动学习**
  - 观察玩家习惯,自动调整提示策略
- [ ] **多 Agent 协作**
  - 一个负责战斗,一个负责剧情,一个负责收集
- [ ] **情感绑定**
  - 长期陪伴产生"羁绊值"
  - 解锁特殊对话/奖励
