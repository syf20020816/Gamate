# Gamate - 项目路线图

> **让每一次游戏都不再孤单** - 实时语音交互 + 智能游戏指导 + 直播场景模拟

## 愿景

打造全球首款开源的智能游戏陪玩 HUD 工具，通过 AI 语音交互、RAG 知识库检索、直播模拟等功能，为单机玩家提供沉浸式的陪伴体验。不挡画面，成本透明，随你定制。

---

## v0.1.0 已完成功能 (2026-02-11) 

### 核心架构
- [x] **Tauri 2.0 + React 19** 桌面应用框架
- [x] **Rust + TypeScript** 前后端分离架构
- [x] **Zustand** 状态管理系统
- [x] **统一配置服务** - `settingsService.ts` 集中管理所有配置
- [x] **Snake_case ↔ CamelCase** 自动转换

### AI 语音交互
- [x] **阿里云 ASR** - WebSocket 实时语音识别（一句话识别）
- [x] **持续监听模式** - VAD 语音活动检测 + 自动端点识别
- [x] **Windows TTS** - 本地语音合成播报
- [x] **5 种 AI 角色** - 损友男/搞笑女/Kobe/甜妹/特朗普（TOML 配置）
- [x] **多模态对话** - 语音+截图联动，AI 结合画面理解上下文
- [x] **Ollama/OpenAI 集成** - 支持本地模型（qwen2.5-vl）和云端 GPT-4V

### 游戏知识库
- [x] **Fandom Wiki 爬虫** - 自动爬取游戏 Wiki 内容
- [x] **向量数据库** - Qdrant 本地部署 + fastembed 向量化
- [x] **RAG 检索** - 百毫秒级语义搜索
- [x] **Skill Library 管理** - 支持多版本、自定义存储路径
- [x] **Steam API 集成** - 自动读取游戏库，识别已安装游戏
- [x] **Steam 游戏自动保存** - 下载的 Steam 游戏自动写入 `games.toml`

### HUD 浮窗系统
- [x] **AI 助手 HUD** - 极简透明浮窗，实时显示 AI 状态
- [x] **直播模拟 HUD** - 独立弹幕窗口，模拟直播间互动
- [x] **跨窗口通信** - Tauri Event 系统实现主窗口与 HUD 同步
- [x] **快捷键支持** - 全局热键唤醒/隐藏 HUD

### 直播场景模拟
- [x] **AI 虚拟观众** - 最多 5 个独立性格的 AI 员工
- [x] **模拟弹幕系统** - 根据游戏进程触发互动（开播/游玩/高光）
- [x] **场景驱动对话** - 不同阶段不同互动策略
- [x] **实时截图分析** - 智能混合定时策略（活跃期 1-15s / 闲置期 15s）

### 游戏管理
- [x] **游戏库管理** - 支持手动添加和 Steam 导入
- [x] **技能库下载** - Wiki 数据下载 + 向量化存储
- [x] **多版本管理** - 支持多个 Wiki 版本切换
- [x] **自动识别** - Steam AppID 自动映射 Wiki 地址

### 其他功能
- [x] **系统托盘** - 最小化到托盘，快捷菜单控制
- [x] **暗色主题** - Ant Design 暗色 UI
- [x] **日志系统** - 前端+后端完整日志记录
- [x] **错误处理** - 统一 Result<T, String> 错误处理

## v0.2.0 计划 (2026-02 ~ 2026-03)

**主题**: 用户体验优化 + 稳定性增强

### 关键功能

#### 1. 语音交互增强
- [ ] **阿里云 TTS 替代 Windows TTS**
  - 支持更自然的音色（甜美女声、磁性男声等）
  - 可调节语速、音量、音调
  - 预期: 提升 AI 陪伴真实感 50%
- [ ] **语音唤醒词**
  - 支持"嘿，Gamate"唤醒
  - 本地轻量级唤醒词检测（< 50MB 模型）
- [ ] **情绪识别**
  - 分析玩家语音情绪（沮丧/兴奋/平静）
  - AI 根据情绪调整回应风格
- [ ] **多轮对话记忆**
  - 短期记忆：记住最近 5 轮对话
  - 长期记忆：保存重要游戏进度（存档关联）

#### 2. 智能截图优化
- [ ] **场景识别**
  - 使用 CV 模型识别游戏场景（战斗/探索/剧情）
  - 自动调整截图频率和 AI 提示策略
- [ ] **OCR 集成**
  - 提取游戏内文字（任务描述、物品说明）
  - 结合 RAG 提供更精准的答案
- [ ] **截图压缩**
  - 发送到 LLM 前自动压缩图片（目标 < 500KB）
  - 减少 API 调用成本 30-50%

#### 3. 性能优化
- [ ] **内存占用优化**
  - 目标：空闲时 < 200MB，对话时 < 400MB
  - 优化向量库加载（延迟加载 + 缓存策略）
- [ ] **启动速度**
  - 冷启动 < 3s（当前约 5-7s）
  - 预加载关键模块
- [ ] **异步优化**
  - 所有 I/O 操作完全异步化
  - 减少 UI 卡顿

#### 4. 错误处理 & 日志
- [ ] **友好错误提示**
  - 所有错误显示中文友好提示
  - 提供具体解决方案（如 API Key 错误）
- [ ] **日志导出**
  - 支持导出最近 7 天日志到文件
  - 方便用户反馈 Bug 时附带日志
- [ ] **崩溃恢复**
  - 自动保存对话历史
  - 崩溃后重启恢复到上次状态

### 用户体验

#### 5. 引导流程优化
- [ ] **首次启动向导**
  - 引导配置 API Key（阿里云 + OpenAI/Ollama）
  - 演示核心功能（语音对话、Wiki 查询、HUD）
- [ ] **快速开始模板**
  - 预置 3 个热门游戏（Phasmophobia/Elden Ring/BG3）
  - 一键下载 Skill Library
- [ ] **视频教程**
  - 录制 3-5 分钟使用教程
  - B 站 + YouTube 发布

#### 6. 设置页面增强
- [ ] **配置验证**
  - 测试 API Key 是否有效
  - 测试向量数据库连接
- [ ] **一键重置**
  - 支持重置到默认配置
  - 清理缓存和临时文件
- [ ] **主题切换**
  - 支持亮色主题（当前仅暗色）

### 质量保证

#### 7. 测试覆盖
- [ ] **单元测试**
  - Rust 后端关键模块测试覆盖率 > 70%
  - TypeScript 工具函数测试
- [ ] **集成测试**
  - 端到端测试（语音输入 → AI 回复 → TTS 播报）
- [ ] **压力测试**
  - 长时间运行稳定性（> 4 小时连续对话）
  - 内存泄漏检测

#### 8. Bug 修复
- [ ] 修复 AI Assistant 游戏选择过滤逻辑（已部分完成）
- [ ] 修复 Steam 游戏图标加载失败问题
- [ ] 修复直播模拟 HUD 窗口偶尔卡死
- [ ] 修复向量数据库查询空结果时崩溃
